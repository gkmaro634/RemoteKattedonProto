# 🐟 ぷるぷるゲンゲ (Puru-Puru Genge)

ヘンテコな地物のおさかな「ゲンゲ」をつついて、コラーゲンたっぷりの体をぷるぷるさせるクリックゲームです。

## 🌟 ゲームの概要
このゲームは、15秒間でどれだけゲンゲを「ぷるぷる」させられるかを競うストイックなクリックゲームです。
ずっと見てたら可愛く見えてくる、かも、しれません。
たくさんぷるぷるして高得点を目指しましょう。

## 🕹️ 遊び方
1. `main.py` を実行します。
2. 制限時間15秒以内に、中央のゲンゲをたくさんクリックしてください。
3. 終了後、スコアに応じた称号が表示されます。
4. 「もう一度ぷるぷる」ボタンで何度でも挑戦可能です。

## 🛠️ 技術構成
- **言語**: Python 3.13+
- **ライブラリ**: Pygame
- **主な機能**: 
  - 画像の動的スケーリングによるアニメーション
  - ファイルI/Oによるハイスコアの永続化
  - 座標判定による正確な当たり判定
